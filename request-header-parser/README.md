#### Functionalities
1. A request to `/api/whoami` should return a JSON object with three fields:

`{
 "ipaddress":"",
 "language":"",
 "software":""
}`

2. Example:

`{
"ipaddress":"159.20.14.100",
"language":"en-US,en;q=0.5",
"software":"Mozilla/5.0 (X11; Ubuntu; Linux x86_64; rv:50.0) Gecko/20100101 Firefox/50.0"
}`
